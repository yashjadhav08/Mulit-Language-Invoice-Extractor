Developed a Streamlit app using Gemini 2.5 Flash to extract and interpret data from multi language invoice images. Applied prompt engineering, image preprocessing, and secure API handling with python dotenv. Built an end-to-end ML tool combining image processing, LLM integration, and interactive UI.

*Steps to run the py file *
Install requirements from requirements.txt
Create an .env file where mention your API key i.e GOOGLE_API_KEY=your_google_api_key_here
Upload invoice images (.jpg, .jpeg, .png)
Ask any question related to the invoice (e.g., "What is the total amount?")
Supports multiple languages
Uses Gemini Vision AI to extract answers from images
Simple and user-friendly interface built with Streamlit
